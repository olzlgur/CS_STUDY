Host: 컴퓨터 + 네트워크 연결 
- 네트워크 이용주체(End-point), 네트워크 자체(Switch)  두 가지로 구성되어 있음
- 네트워크 이용주체(End-point) ex ) Peer, Server, Client
- 네트워크 자체(Switch) ex) Router ( 경로 선정 목적 ), Fire Wall, IPS ( 보안 목적 )

Network
→ Internet ( Router + DNS )
